# ActiveCampaignRails

Simple rails wrapper for ActiveCampaign API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'active-campaign-rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install active-campaign-rails
